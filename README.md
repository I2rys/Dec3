# Dec3
A really fast, reliable & accurate MD5, SHA1, SHA224, SHA256, SHA384, SHA512 and RIPEMD320 decryptor.

## Installation
npm:

    npm i chalk && npm i dcipher

# Usage

    node index.js <MD5/SHA1/SHA224/SHA256/SHA384/SHA512/RIPEMD320>
